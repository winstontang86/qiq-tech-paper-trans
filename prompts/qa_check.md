# 质检规则（阻断级）

`qa_report.py` 对翻译产物执行以下检查。**任一阻断项不通过，流程终止并输出 `qa.md` 报告**；除非用户显式传入 `--force`。

## 阻断级（Blocker，default stop）

| 编号 | 检查项 | 阻断条件 |
|---|---|---|
| B1 | 段落数对齐 | 译文段落数 < 原文段落数 |
| B2 | 图片保留 | `![...](...)` 的数量少于原文 |
| B3 | 表格保留 | Markdown 表格块数量少于原文 |
| B4 | 公式保留 | `$...$` / `$$...$$` / `\\begin{equation}` 块数量少于原文 |
| B5 | 代码块保留 | fenced code block 数量少于原文 |
| B6 | 引用编号保留 | `[\\d+]` / `\\(Author, Year\\)` 数量少于原文 |
| B7 | 段级长度比 | `len(zh_chars)/len(en_chars)` 不在 [0.8, 2.5]（经验区间） |
| B8 | 摘要性短语 | 译文出现"综上所述 / 简而言之 / 总的来说 / 作者主要讨论了 / 简要介绍 / 下面我们" 且原段未出现对应英文 |
| B9 | 锁定块完整性 | 占位符回贴后，锁定块哈希与原文不一致 |
| B10 | References 未翻译 | References / Bibliography 节内容与原文字节级一致 |

### 长度比区间说明

中译英/英译中的经验比率：
- 英→中字符比例通常在 1.0–2.0 之间
- 允许上限放到 2.5 容纳长术语"中文（English）"格式
- 下限 0.8 捕获被压缩/摘要的段落

## 警告级（Warn，仅提示不阻断）

| 编号 | 检查项 |
|---|---|
| W1 | 术语一致性：同一英文术语对应多个中文译法 |
| W2 | 中英文之间缺少空格 |
| W3 | 中文段落混用半角标点 |
| W4 | 疑似未翻译（段落仍为纯英文） |

## QA 报告结构

```markdown
# 翻译质检报告

- 输入：<path>
- 输出：<path>
- 时间：<timestamp>
- 总段数：N（原文）/ M（译文）
- 阻断项：X 项 / 警告项：Y 项
- 结论：通过 / 阻断

## 阻断项详情
### B4 公式保留异常
- 原文公式数：23
- 译文公式数：21
- 缺失段落：seg_042, seg_067

## 警告项详情
...

## 段级详情
| seg_id | 原文字符 | 译文字符 | 长度比 | 状态 |
...
```

## 跳过机制

- `--force`：跳过所有阻断项，仅生成警告报告
- `--skip-checks B7,B8`：跳过特定检查项（逗号分隔）
